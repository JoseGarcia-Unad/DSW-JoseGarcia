¿ Que es Frameworks CSS y posicionamiento Web?

Actualmente la educación virtual se ha convertido en una gran herramienta para la continuidad del aprendizaje,
es así que se hace estrictamente necesario y de manera urgente la interacción con las diferentes plataformas
virtuales y herramientas ofimáticas. En este sitio web se definirá los conceptos sobre Frameworks CSS y el
posicionamiento web con el fin de conocer sus principales características y su utilidad en la informática.

Frameworks CSS

Un Frameworks es una estructura conceptual y tecnológica que puede servir como base para la organización y desarrollo
de un software, siendo un punto de partida para la elaboración de proyectos con objetivos claramente definidos. Alguna
de sus principales características son:

  1. Tiene una estructura previa que se puede utilizar para el desarrollo de proyectos facilitado el trabajo.
  2. Se puede entregar el proyecto en menos tiempo y con un código más limpio.
  3. Se logra hacer cosas más avanzadas que un desarrollador no haría o que le costaría mucho tiempo hacerlo.
  4. Tiene funciones y herramientas ya desarrolladas.
  5. Facilita el trabajo en equipo, pues al haber trabajado con marcos de desarrollo estandarizados, los demás
    programadores entenderán el proyecto más fácil.
Systems, O. (2018, 11 enero). ¿Qué es un framework y para qué se utiliza? Orix Systems.
https://www.orix.es/que-es-un-framework-y-para-que-se-utiliza


Posicionamiento web

El posicionamiento web o posicionamiento en buscadores consiste en el conjunto de acciones orientadas a mejorarb
la ubicación de un sitio web en los diferentes buscadores en internet mejorando el contenido con el objetivo de
volverlos más útiles y relevantes para el usuario.
En la actualidad existe la tendencia de negocios o ventas on line, entonces se hace estrictamente necesario que la
gente conozca de ti y de los productos que ofrece y esto se puede lograr aplicando el conjunto de acciones de SEO.

Para una buena posición de tu web y lograr mayor ventas o visitas a tu página web se puede aplicar las siguientes
acciones:

SEO on-page: Consiste en las técnicas aplicadas dentro de la misma página como :

  • Mejoras de la velocidad y optimización de la web
  • Optimización de imágenes
  • Creación de contenido de calidad
  • Buenas experiencia de usuario web
  • Buena estructura de las etiquetas HTML

SEO off-page: Consiste en las técnicas aplicadas por fuera de la página como:
  • Uso de redes sociales
  • Estrategias de link building
  • Guest blogging
  • Registro en direcciones
